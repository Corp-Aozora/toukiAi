１ 定義
  1. 本サービス：当社が運営するサービス及び関連するサービス
  2. 本サイト：本サービスのコンテンツが掲載されたWebサイト
  3. 本コンテンツ：本サービスが提供する情報、ソフトウェアプログラム、コード等の総称
  4. 登録ユーザー：本サービスのアカウント登録が完了した方
  5. ＩＤ：登録ユーザーの識別番号
  6. 個人情報：住所、氏名、電話番号等個人を特定することのできる情報の総称
  7. 登録情報：登録ユーザーが本サイトにて登録した一切の情報
２ 本規約への同意
  1. 登録ユーザーは、本利用規約及びアカウント登録前に提示される利用条件に同意したものとみなします。
  2. 本サービスのアカウント登録をした時から本規約が登録ユーザーと当社の間に本規約が適用されます。
  3. 未成年者が利用する場合は、親権者その他の法定代理人の同意を得てください。
  4. 未成年者が、法定代理人の同意がないにもかかわらず同意があると偽りまたは成年と偽って本サービスを利用した場合、本サービスに関する一切の法律行為を取り消すことは出来ません。
  5. 本規約の同意時に未成年であった登録ユーザーが成年に達した後に本サービスを利用した場合、当該登録ユーザーは本サービスに関する一切の法律行為を追認したものとみなされます。
３ 規約の変更
  1. 当社は、登録ユーザーの承諾を得ることなく、いつでも、本規約の内容を改定することができるものとし、登録ユーザーはこれを異議なく承諾するものとします。
  2. 当社は、本規約を改定するときは、その内容について当社所定の方法により登録ユーザーに通知します。
  3. 本規約の改定の効力は、当社が前項により通知を行った時点から生じるものとします。
  4. 登録ユーザーは、本規約変更後、本サービスを利用した時点で、変更後の本利用規約に異議なく同意したものとみなされます。
４ アカウント登録
  1. 本サービスの利用を希望する方は、本規約及びアカウント登録前に提示される利用条件に同意したうえで登録してください。
  2. 本サービスの利用を申し込んだ方は、弊社からアカウント登録完了の通知が届いた時から登録ユーザーとなります。
  3. 当社は、登録ユーザー向けにメールで本サービスに関連する連絡することがありますので、あらかじめご了承ください。
５ アカウント登録の拒否、取り消し
  1. 過去に本規約または当社が定めるその他の定めに違反したことを理由として退会処分を受けた者であるとき、利用申込を拒否することがあります。
  2. 当社が定める方法によらずアカウント登録したとき、その登録を取り消すことがあります。
  3. その他当社が不適切と判断した場合、利用申込を拒否し、または登録を取り消すことがあります。
６ アカウントの管理
  1. 登録ユーザーは、第三者に自己のアカウントを利用させ、または貸与、譲渡、名義変更、売買などをしてはならないものとします。
  2. 前項により当社または第三者に損害が生じた場合、登録ユーザーは当社および第三者に対して、当該損害を賠償するものとします。
  3. 登録ユーザーの個人情報が不正確または虚偽であったために登録ユーザーが被った一切の不利益および損害に関して、当社は責任を負わないものとします。
  4. 登録ユーザーの個人情報が盗用されまたは第三者に利用されていることが判明した場合、登録ユーザーは直ちにその旨を当社に通知するとともに、当社からの指示に従うものとします。

当社が別途定めるプライバシーポリシーに準拠して適切に取り扱います。
８ 禁止行為
  1. 当社は、本人から個人情報が利用目的の範囲を超えて取り扱われているという理由、または不正の手段により取得されたものであるという理由により、その利用の停止または消去（以下、「利用停止等」といいます。）を求められた場合には、遅滞なく必要な調査を行います。
  2. 前項の調査結果に基づき、その請求に応じる必要があると判断した場合には、遅滞なく、当該個人情報の利用停止等を行います。
  3. 当社は、前項の規定に基づき利用停止等を行った場合、または利用停止等を行わない旨の決定をしたときは、遅滞なく、これをユーザーに通知します。
  4. 前２項にかかわらず、利用停止等に多額の費用を有する場合その他利用停止等を行うことが困難な場合であって、ユーザーの権利利益を保護するために必要なこれに代わるべき措置をとれる場合は、この代替策を講じるものとします。
